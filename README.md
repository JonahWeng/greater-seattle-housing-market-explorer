# Greater Seattle Housing Market Explorer

Public interactive website for comparing city-level housing market indicators across 78 Greater Seattle cities.

Live site: https://jonahweng.github.io/greater-seattle-housing-market-explorer/

## Included

- Median home price and price-per-square-foot changes
- Single-family, townhouse, and condo/co-op comparisons
- 1-year, 3-year, and 5-year windows
- Seasonal peak comparisons
- Population, inventory, months of supply, and sales metrics
- City-scope and low-sample controls
- Mouse, button, and touch zoom

## Data sources

Redfin Data Center, Washington State Office of Financial Management, and U.S. Census Bureau.

Only the public presentation artifact is stored in this repository. Internal workbooks, source snapshots, and processing files are intentionally excluded.
